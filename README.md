# SpectroIBIS
**SpectroIBIS** is a user-friendly desktop application designed to automate tedious tasks in the analysis of multi-conformer quantum chemical spectroscopic calculations, such as for predictions of ECD spectra or NMR chemical shifts of natural products.


## Key features
+ Extracts data from Gaussian or ORCA output files

+ Checks for redundant conformer geometries and imaginary frequencies

+ Saves Boltzmann-averaged spectroscopic data (ECD, VCD, UV-Vis, IR, optical rotation or NMR) in convenient formats, such as BIL files for SpecDis

+ Generates publication-quality supplementary data tables with conformer energies and atomic coordinates

+ Can also create input files for Gaussian or ORCA

+ Simple drag-and-drop user interface
